Gamer chat app!

Our first major project at UIT was to build a instant messaging app using HTML, CSS, Javascript and Firebase. That was the first time I have ever built something with code from scratch. While buidling this app I became very familiar with HTML and CSS since before that I've only completed online courses. Since I really enjoy video gaming, I designed my chat app for gamers. The app has different chat rooms for different games and also has a gaming news feed section.
